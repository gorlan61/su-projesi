# 💧 Premium Su Satış ve Akıllı Takip Platformu

Modern web teknolojileri kullanılarak geliştirilmiş, kullanıcılara sadece su satışı yapmakla kalmayan, aynı zamanda **tüketim alışkanlıklarını takip eden** interaktif bir e-ticaret platformudur. 

Proje, herhangi bir backend kullanılmadan **React Context API** ve **LocalStorage** mimarisiyle state yönetimini (veri kalıcılığını) istemci tarafında (client-side) profesyonelce simüle eder.

## ✨ Öne Çıkan Özellikler

* 📊 **Akıllı Tüketim Asistanı:** Kullanıcının seçtiği profile (Örn: Haftalık 2 Damacana) göre evdeki su miktarını hesaplar ve bitmeye yakın uyarı verir.
* ⚖️ **Dinamik Ürün Karşılaştırma:** Kullanıcıların farklı su markalarını (PH, Mineral, Fiyat) yan yana kıyaslamasına olanak tanır.
* 🚚 **Kişiselleştirilmiş Teslimat:** "Zile Basma", "Temassız Teslimat" gibi kurye tercihleri LocalStorage üzerinde tutularak sonraki siparişlerde otomatik hatırlanır.
* ♻️ **Depozito Sihirbazı:** Farklı marka damacana değişimlerinde sepet tutarını dinamik olarak günceller.

## 🛠️ Kullanılan Teknolojiler (Tech Stack)

* **Frontend Framework:** React.js
* **State Management:** Context API (OOP prensiplerine uygun kapsüllenmiş yapı)
* **Styling:** Tailwind CSS (Modern, responsive ve cam efekti tasarımlar)
* **Veri Kalıcılığı:** LocalStorage API
* **Routing:** React Router DOM
